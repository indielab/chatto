## ServerService

Provides authenticated server profile, runtime configuration, and public
server member reads.

<a id="chatto-api-v1-ServerService-GetServerState"></a>

### GetServerState

Returns authenticated server state. This RPC requires a logged-in user;
public discovery remains available through
chatto.discovery.v1.ServerDiscoveryService.GetServer.

```http
POST /api/connect/chatto.api.v1.ServerService/GetServerState
```

<a id="chatto-api-v1-GetServerStateRequest"></a>

#### Input: GetServerStateRequest

Request for authenticated server state.


<a id="chatto-api-v1-GetServerStateResponse"></a>

#### Result: GetServerStateResponse

Authenticated server state used by the web client and public API clients.

| Field | Type | Description |
| --- | --- | --- |
| `profile` | [`ServerProfile`](/reference/connectrpc-api/types/#chatto-api-v1-ServerProfile) | Authenticated server profile and branding. |
| `runtime` | [`ServerRuntimeConfig`](/reference/connectrpc-api/types/#chatto-api-v1-ServerRuntimeConfig) | Authenticated runtime settings used by clients. |


<a id="chatto-api-v1-ServerService-ListMembers"></a>

### ListMembers

Lists authenticated server members. Every authenticated user is a server
member; admin-sensitive fields stay out of this public row shape.

```http
POST /api/connect/chatto.api.v1.ServerService/ListMembers
```

<a id="chatto-api-v1-ListServerMembersRequest"></a>

#### Input: ListServerMembersRequest

Request for server members visible to the authenticated user.

| Field | Type | Description |
| --- | --- | --- |
| `search` | `string` | Optional case-insensitive search against login and display name. |
| `page` | [`PageRequest`](/reference/connectrpc-api/types/#chatto-api-v1-PageRequest) | Page request. Defaults to 20 results when absent or limit is zero. |


<a id="chatto-api-v1-ListServerMembersResponse"></a>

#### Result: ListServerMembersResponse

Server member page.

| Field | Type | Description |
| --- | --- | --- |
| `members` | repeated [`DirectoryMember`](/reference/connectrpc-api/types/#chatto-api-v1-DirectoryMember) | Members in the requested page. |
| `page` | [`PageInfo`](/reference/connectrpc-api/types/#chatto-api-v1-PageInfo) | Page metadata. |
